JavaScript 運算符類型

算術運算符

用於數字的算術計算。

運算符描述例子
+加法x + y
-減法x - y
*乘法x * y
/除法x / y
%求餘x % y
**指數x ** y
++自增x++
--自減x--

賦值運算符

用於將值賦予變量。

運算符例子等同於
=x = yx = y
+=x += yx = x + y
-=x -= yx = x - y
*=x *= yx = x * y
/=x /= yx = x / y
%=x %= yx = x % y
**=x **= yx = x ** y

比較運算符

用於比較兩個值。

運算符描述例子
==等於x == y
===值和類型都相等x === y
!=不等於x != y
!==值或類型不等x !== y
>大於x > y
<小於x < y
>=大於等於x >= y
<=小於等於x <= y

邏輯運算符

用於基於兩個值之間的邏輯關係決定表達式的真偽。

運算符描述
&&邏輯與
||邏輯或
!邏輯非

位元運算符

操作整數的二進制位。

運算符描述例子結果
&AND5 & 11
|OR5 | 15
~NOT~510
^XOR5 ^ 14
<<左移5 << 110
>>右移5 >> 12
>>>無符號右移5 >>> 12

三元運算符

一個條件運算符，是 JavaScript 中唯一擁有三個操作數的運算符。

語法: 條件 ? 表達式1 : 表達式2

例子: let result = age > 18 ? "成年" : "未成年";

類型運算符

用於返回變量或表達式的數據類型。

例子: typeof x 返回 x 的類型。

例子: x instanceof Array 檢查 x 是否為 Array 的實例。
