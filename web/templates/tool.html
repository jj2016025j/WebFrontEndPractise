To quickly open a file, use the kbd element:

Use Ctrl + P to open the quick file search.

To open the command palette, use the kbd element:

Use Ctrl + Shift + P to open the command palette.

To save the current file, use the kbd element:

Use Ctrl + S to save the current file.

To create a new file, use the kbd element:

Use Ctrl + N to create a new file.

To close the current editor window, use the kbd element:

Use Ctrl + W to close the current editor window.

To cut the selected text, use the kbd element:

Use Ctrl + X to cut the selected text.

To copy the selected text, use the kbd element:

Use Ctrl + C to copy the selected text.

To paste the copied text, use the kbd element:

Use Ctrl + V to paste the copied text.

To undo the last action, use the kbd element:

Use Ctrl + Z to undo the last action.

To redo the last undone action, use the kbd element:

Use Ctrl + Y to redo the last undone action.
